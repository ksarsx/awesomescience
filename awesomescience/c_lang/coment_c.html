Комментарии в Си


Как и вдругих языках программирования в языке Си можно и нужно писать комментарии. Они помогают легче ориентироваться в коде. Допустим вы открываете проект спустя неделю, скорее всего вы забудите логику многих функций, которые вы используете, в этом случае коментарии помогают быстро разобраться в коде, это также полезно для чтения кода другими программистами, если ты работаешь над проектом в команде.

Однострочные комментарии

Однострочные комментарии могут находиться в любой части кода, записываются при помощи двух слешей: //.
С 
#include <stdio.h>
int main() {
// однострочный комментарий
return 0;        
}

Многострочные комментарии

Многострочные комментарии также как и однстрочные, можно писать где угодно. Они записываются при помощи открывающей комментарий /* и закрывающей */, текст, находящийся между /* и */ считается комментарием и не читается компилятором.
С 
#include <stdio.h>

/*
Многострочный 
комментарий
*/

int main() {
    return 0;
}
