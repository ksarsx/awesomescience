Логические значения и условные операторы

Как такового логического типа данных bool в языке Си нет. Однако его логику реализуют с помощью 1 или 0. Также следует учитывать, что правдой считается любое значение не равное нулю. Для получения логических значений используем условные операторы, они используются для сравнения значений, в случае, если сравнение верно, то они возращают правду, иначе ложь.

Условные бинарные операторы:
  • == - возвращает правду, когда оба сравниваемые значения совпадают.
  • != - возвращает правду, когда оба сравниваемые значения не совпадают.
  • > - возвращает правду, когда значение слева больше чем значение справа.
  • < - возращает правду, когда значение слева меньше чем значение справа.
  • >= - возвращает правду, когда левое значение больше либо равно правому.
  • <= - возращает правду, когда левое значение меньше либо равно правому.
И условный унарный оператор ! инвертирует значение, то есть из правды в ложь, из лжи в правду.
C 
#include <stdio.h>

int main() {
    int a = 1 > 0;
    printf("%d", a);
    return 0;
}
Вывод 
1

С 
#include 

int main() {
    int a = !46;
    printf("%d", a);
    return 0;
}

#include 0
