Переменные и типы данных


Переменную можно назвать контейнером или небольшой коробкой, в которой хранятся какие-то данные. Этими данными может быть число, символ или указатель. Под каждый тип данных существует свой тип переменной (контейнера).
В этой статье будет использован scanf для вывода результата программ, его объяснение будет в следующей статье функции printf и scanf.

Типы данных

int

Это целочисленный тип данных, то есть в переменную этого типа можно положить целове число. Переменные этого типа занимают 4 байта памяти (имеют размах от -32768 до 32767). Вот пример создания переменной типа int:

C 
#include <stdio.h>

int main() {
    /*присваиваем в перменную с названием number 
    целочисленное значение равное 10*/
    int number = 10;                        
    return 0;
}


Мы можем складывать значения переменных, умножать, делить, брать остаток от деления и проводить другие арифметические операции со значениями переменных, а потом присваивать результаты этих операций в сами переменные:

C 
#include <stdio.h>

int main() {
    int a = 10;
    int b = 12;
    int c = a + b;
    printf("%d", c);                        
    return 0;
}

float

Это вещественный тип данных (с плавающей запятой). Занимает столько же памяти как и int - 4 байта. Подходит для работы с дробными и иррациональными числами.

С 
#include <stdio.h>
              
int main() {
  float a = 2.5;
  float b = a;
  printf("%f", a + b);
  return 0;
}

Вывод 
5.000000
C:\Project21\x64\Debug\Project21.exe (процесс 15232) завершил работу с кодом 0 (0x0).


double

Это тоже вещественный тип данных (с плавающей запятой). Но занимает уже 8 байт памяти. Отличиается от float только тем, что под него выделяется больше памяти.

char

Тип данных, который позволяет хранить один символ. Занимает 1 байт.

Здесь просто выводим один символ
С 
#include <stdio.h>
int main() {
  char sym = 'A';
  printf("%c", sym);
  return 0;
}

Символы в си хранятся как коды в таблице аски, поэтому при переводе в целочисленный вид мы получаем коды символов.
С 
#include <stdio.h>
int main() {
  char c = '0';
  printf("%d", c); //выводим в интовом виде
}

Вывод 
48
C:\Project21\x64\Debug\Project21.exe (процесс 6656) завершил работу с кодом 0 (0x0).

int main() {

  printf("%d", number);
}
