Инкременты и декременты

Инкремент

Это оператор, который увеличивает значение переменной на 1. Пишется как ++ после в постфиксной форме и перед в префиксной.
C 
#include <stdio.h>

int main() {
    int n = 0;
    n++;
    printf("%d", n);
}
Вывод 
1

Префиксный инкремент сначал меняет значение, а потом возвращает его. Вот так:
C 
#include <stdio.h>

int main() {
    int x = 0;
    int y = ++x;
    printf("%d", y);
}
Вывод 
1

Постфиксный инкремент сначала возвращает значение, а потом меняет его.
C 
#include <stdio.h>

int main() {
    int x = 0;
    int y = x++;
    printf("%d", y);
}
Вывод 
0

Декремент

Декремент вместо прибавления единички вычитает её. Записывается с помощью --. А так в префиксной и постфиксной форме работает также.
C 
#include 

int main() {
    int n = 0;
    n--;
    printf("%d", n);
}
Вывод 
-1
