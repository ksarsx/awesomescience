Локализация


Локализация в нашем случае нужна для корректного вывода в консоль. То есть без неё символы на русском языке например будут выглядеть как непонятные крокозябры.
C 
#include <stdio.h>

int main() {
    printf("Привет, Мир!");
    return 0;
}
Вывод 
╧ЁштхЄ, ╠шЁ!

Чтобы поправить данную ситуацию импортируем locale.h и пользуемся функцией setlocale:
C 
#include <stdio.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "ru");
    printf("Привет, Мир!");
    return 0;
}

Вывод 
Привет, Мир!
