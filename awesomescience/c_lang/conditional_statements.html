Условные конструкции


if

if или "если" - это условная конструкция, которая выполняет код внутри своего тела, если парметр, который она принимает имеет истинностное значение. 
#include <stdio.h>
if (условие) {
    код, который выполняется при соблюдении условия
}

Вот примитивный пример робаты if:
C 
#include <stdio.h>
#include <locale.h>

int main() {
    setlocale(LC_ALL, "ru");
    
    if (1 > 0) {
        printf("Один больше нуля");
    }

    return 0;
}

Вывод 
Один больше нуля

else if

else if или "иначе если" - обычно стоит после if или else if, важно заметить, что если перед ним стоящий if или else if выполнится, то до него дело не дойдет. Это тоже условная конструкция, которая выполняет код внутри своего тела, в случе если парметр имеет истинное значение.
C 
#include <stdio.h>
#include <locale.h>

int main() {
  setlocale(LC_ALL, "ru");
  
  if (1 > 2) {
    printf("Ложь");
  }
  else if (2 > 1) {
    printf("Правда");
  }
  
  return 0;
}
Вывод 
Правда

else

else или иначе - условная конструкция, которая следует после if или else if и выполняется в случае, если ничто до него не вполнилось.
C 
#include <stdio.h>
#include <locale.h>

int main() {
  setlocale(LC_ALL, "ru");
  
  if (1 > 2) {
    printf("Ложь");
  }
  else if (1 == 2) {
    printf("Ложь");
  }
  else {
    printf("Все предыдущие условия - ложные");
  }
  
  return 0;
}
Вывод 
Все предыдущие условия - ложные
