printf и scanf


Функция printf

Функция printf предназначена для вывода (в основном в консоль) отформатированного текста. Первый параметр, который она принимает - это строка формата, в ней может содержаться как просто текст, так и специальные шаблоны - типы аргументов, которые идут после первого аргумента (форматной строки). В местах, где подставлены эти шаблоны как раз и будут выведены значения аргументов после форматной строки.

Шаблоны:
  • %d - для вывода целочисленных значений
  • %f - для вещественных значений
  • %с - для вывода символа
  • %p - для вывода адресов
С 
#include <stdio.h>

int main() {
    printf("%d", 10);
    return 0;
}

Вывод 
10

Функция scanf

Функция scanf является обратной функцией к функции printf, делает тоже самое, только наоборот. Тоесть считывает значения из консоли. Шаблоны у нее те же. Аргументы, которые идут после шаблонной строки, должны быть переменными, перед ними мы пишем & - это оператор взятия адреса, в которые будут переданы значения с консоли.

Далее пример программы, которая считывает значения 2 значения и выводит их сумму.
С 
#include 

int main() {
    int a;
    int b;

    scanf("%d %d", &a, &b);

    int sum = a + b;

    printf("%d", sum);

    return 0;
}

ввод 
35 60

вывод 
95
